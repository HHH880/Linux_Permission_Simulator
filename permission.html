how the math works
Each permission triplet rwx maps to a 3-bit binary number: r = 4 (100), w = 2 (010), x = 1 (001). Add the values of enabled bits to get the octal digit.

rwx = 4+2+1 = 7  |  r-x = 4+0+1 = 5  |  rw- = 4+2+0 = 6  |  --- = 0
Three triplets give three digits: chmod 755 means owner=7, group=5, others=5.

useful chmod commands
$chmod 755 script.sh      # make executable
$chmod 600 ~/.ssh/id_rsa   # private key
$chmod -R 755 /var/www/    # recursive
$chmod a+r README.md       # symbolic — add read for all
$chmod go-w config.conf    # remove write from group+others
$ls -l filename             # view current permissions
$stat filename              # detailed permission info
